## Description
A modern, glassmorphism-based weather web application built using React and Vite. It fetches real-time weather data and displays it with a clean, dynamic UI.

## Features
- Real-time weather forecasting using Open-Meteo API
- Clean glassmorphism UI design
- Dynamic weather icons
- Fully responsive

## Setup Instructions
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
